# Phase 3 — Create the folder structure

Inside the voice-rag-assistant project, create the following empty folders so the codebase is organized before writing any logic: src/lib/supabase, src/lib/rag, src/lib/vapi, src/components/upload, src/components/documents, src/components/voice, src/components/chat, src/hooks, src/store, src/types, sample-documents (at the project root), and the API route folders src/app/api/upload, src/app/api/ingest, src/app/api/documents, src/app/api/query, and src/app/api/vapi/webhook.
